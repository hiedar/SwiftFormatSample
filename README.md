# SwiftFormat Sample

## Prerequisite

* macOS  
High Sierra 10.13.6
* Xcode  
10.1

## Configuration

1. [Install rbenv](https://github.com/rbenv/rbenv)
2. `rbenv install 2.6.2`
3. `rbenv exec gem install bundler -v "2.0.1"`
4. `rbenv exec bundle install --path .bundle`
5. `rbenv exec bundle exec pod install`
